- Réinitialiser à l'état du dernier commit : 
```html
git reset --hard HEAD 
```

- Changer de branche : 
```html
git checkout nom-branche
```
- Renommer fichier : 
```html
git mv old_filename new_filename
```

- Sauvegarder modifications : 
```html
git stash
```

- Restaurer modifications : 
```html
git stash apply
```

- Consulter la liste des stashs : 
```html
git stash list
```
